# Bank-Loan-Analysis-Dashboard---PowerBI

# Description

Developed an interactive Bank Loan Analysis Dashboard using Power BI to provide actionable insights into loan disbursements, approvals, defaults, and customer trends. The project focused on enabling banking stakeholders to make data-driven decisions for improving loan strategies and minimizing risks.
